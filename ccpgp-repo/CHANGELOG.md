# Changelog

## 1.0.0 (2026-03-25)

### Initial release

- `CCANeuron` — Dual-timescale neuron with narrative-modulated threshold
- `CCPGPSynapse` — Four-factor learning rule (STDP × Constitutional × Provenance × Regret)
- `CCPGPNetwork` — Automatic current routing, constraint checking, classification helper
- `PyramidalCCA`, `FastPVCCA`, `SlowSSTCCA`, `VIPCCA`, `TRNCCA` — Five heterogeneous neuron types
- `HeteroSynapse` — Multi-mode plasticity (ccpgp / fixed / inhibitory_homeostatic)
- `HeteroNetwork` — Heterogeneous network with mixed synapse types
- 12 verification tests (22 assertions), all passing
- Two examples: quickstart and identity divergence
- Zero external dependencies
